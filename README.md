# Java Collections and Exception Handling Task

## Overview
This project demonstrates the use of Java Collections Framework and Exception Handling concepts. It includes examples of `ArrayList`, `HashMap`, `HashSet`, `try-catch` blocks, and a custom exception.

## Features
- ArrayList example
- HashMap example
- HashSet example
- Exception handling using try-catch
- Custom exception implementation

## Project Structure

```
Java-Collections-Exception-Task/
│

├── HashMapExample.java
├── HashSetExample.java
├── ExceptionHandlingExample.java
├── CustomExceptionExample.java
└── README.md
```

```bash
javac *.java
```

## How to Run

```bash
java ArrayListExample
java HashMapExample
java HashSetExample
java ExceptionHandlingExample
java CustomExceptionExample
```

## Learning Outcomes
- Understand Java Collections Framework.
- Manage data using ArrayList, HashMap, and HashSet.
- Handle runtime errors using try-catch.
- Create and use custom exceptions.
